Portfolio

Helicopter Game - Java

This is a helicopter game I made in java. The aim of this game is to shoot down as many enemy helicopters as possible. To add to the challenge, enemies multiply after each wave and speeds vary from enemy to enemy. A health bar, displayed on the top right, shows the players health status. Object oriented programming and design patterns are utilised in this application.

Binary-Decimal Converter - Python

The aim of this program is to convert binary numbers to decimal numbers, and decimal numbers to binary numbers. For binary to decimal conversions, the program accepts 8 bit numbers. For binary to decimal conversion, the program utilises the positional notation methodology. For decimal to binary conversion, short division by two with remainder methodology is used.

Snake Game - Java

This is an adaptation of the classic snake game I made using the java programming language. The aim of this game is to avoid collision with the walls or the snake itself for as long as possible. The program requires the player to use the arrow keys to control the snake's direction.Getting the red square extends the length of the snake by 1 square and also adds 1 point to the score. the score, time and length of the snake is displayed at the top of the screen.

Full code for this website and other projects can be found on my Github page.
